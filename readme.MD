A Java CLI Application for Search.

Requirements
1. JDK 1.8
2. Maven

Steps to Run
1. **mvn clean install** (on the root directory)
2. **cd target/**
3. **java -jar search-system-1.0-SNAPSHOT.jar ../directory**
